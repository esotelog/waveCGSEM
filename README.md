## To run the executable: 
./waveCGSEM param_waveCGSEM.prm

## Output
- dat file of receivers data
- vtk files (optional)
- waveCGSEM_log.txt

## Example folder
Propagation of a shear plane wave originated at the top boundary in a homogeneous medium

### Run the executable 
./waveCGSEM example.prm

## Developer corner
This project uses the [deal.II library](https://dealii.org/) version 9.7.0. The library needs to be installed to compile this project from the source code.
